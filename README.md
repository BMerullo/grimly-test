# Grimly Test Repository

This repository serves as a sandbox for testing features, libraries, and integrations intended for the Grimly app. It is a playground to experiment with tools such as Native Firebase, authentication systems, and other planned functionalities before implementing them in the main project.

## Features Being Tested

- **Firebase Integration**: Testing Native Firebase for authentication, database management, and other backend services.
- **Authentication**: Experimenting with secure user login flows and session handling.
- **React Native Components**: Prototyping and validating UI/UX elements.
- **Third-Party Libraries**: Evaluating tools and plugins for compatibility with the Grimly app.

## Project Structure

- `src/`: Contains all the test components and feature implementations.
- `tests/`: Includes test scripts for various functionalities.

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/grimly-test.git
   ```
2. Navigate into the project directory:
   ```bash
   cd grimly-test
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the project:
   ```bash
   npm start
   ```
